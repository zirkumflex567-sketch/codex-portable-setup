# Errors

Reusable failures, command errors, and integration gotchas.

Use this file when:

- a command fails unexpectedly
- a tool/plugin integration breaks
- the same setup mistake is likely to happen again

## Template

## [ERR-YYYYMMDD-001] name

**Logged**: ISO-8601 timestamp  
**Priority**: low | medium | high | critical  
**Status**: pending | resolved | promoted  
**Area**: workflow | tools | docs | code | infra

### Summary
Short description of the failure.

### Error
```text
Exact error message or the key failing line.
```

### Context
Command, tool, machine, and relevant setup details.

### Suggested Fix
What should be done next time.
